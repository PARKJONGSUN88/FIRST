### 라인 피드, 캐리지 리턴

라인피드(LF : Line Feed)  : 현재 위치에서 아래줄로 이동

캐리지리턴(CR: Carriage return)  : 커서의 위치를 맨 앞으로 이동



### Java에서  줄바꿈

outputstream을 이용하여 텍스트 파일로 출력하였는데  콘솔창과 달리 텍스트 파일에서는 줄바꿈이 되지않았다.

```python
String listFar = listFin + "\n";
```

이유는 줄바꿈시 

**window 는 \r\n**

**유닉스나 리눅스는 \n** 이다.



```python
String listFar = listFin + "\r\n";
```
